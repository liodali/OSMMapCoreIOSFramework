# OSM MapCore iOS Framework
### Overview

 This iOS framework that use MapCore SDK to build OSM Map and simplification of the API

 * current version : `0.6.6`


 #### Installation

 * In `.podspec`
 ```
 spec.dependency "OSMFlutterFramework"
 ```

 * In `podfile`
 ```
 pod "OSMFlutterFramework"
 ```
